# 🔥 Fire Hazard Detection using CNN and SVM

## Overview

This project classifies an image into:

- Fire Hazard
- Non Hazard

using Machine Learning and Deep Learning.

---

## Dataset

Source:

Roboflow Fire Detection Dataset

Images

17,344

Classes

Fire

Smoke

Light

Non Fire

Converted into Binary Classification

Fire Hazard

Non Hazard

---

## Models Used

CNN (MobileNetV2)

SVM

---

## Results

CNN Accuracy

91.94%

SVM Accuracy

96.51%
